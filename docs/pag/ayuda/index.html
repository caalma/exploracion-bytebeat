Ayuda y modo de uso


Los campos códigos disponen de ciertas utilidades interactivas.

  1. Clikeando sobre cualquiera de ellos el código de la misma se copia al portapapeles. Además se activa como código-actual.
  2. Cuando el campo está en foco y con el cursor en modo de edición es posible cambiar el contenido. Al darle click nuevamente se vuelven a realizar las acciones del item 1.
  3. Cuando existe un código-actual activo quedan disponibles algunas utilidades en el márgen derecho de la página.
  4. La utilidad P reproduce o pausa el audio generado mediante el código-actual.
  5. La utilidad T abre la aplicación externa Touchbit con el código-actual, para escucharlo o editarlo allí.
  6. La utilidad H abre la aplicación externa HTML5Bytebeat con el código-actual, para escucharlo o editarlo allí.
  7. El campo numérico junto a cada campo de código especifíca el sample rate con que se escuchará dicho audio. Al modificarlo, automáticamente se actualizan los enlaces de T y H, y además, si el audio se está reproduciendo, será afectado con el nuevo sample rate.
  8. Estando dentro de cualquier campo de código existe tres atajos de teclado:
  9. Alt+p hace lo mismo que la utilidad P,
  10. Alt+t hace lo mismo que la utilidad T,
  11. Alt+h hace lo mismo que la utilidad H,
  12. 'Alt+Enter' hace lo mismo que darle click al campo de código.
